## Подключение ВКонтакте — пошагово

### Шаг 1. Создать приложение VK

1. Перейдите на **https://vk.com/editapp?act=create**
2. Выберите тип: **Standalone-приложение**
3. Введите любое название (например, «Генератор постов»)
4. Нажмите «Подключить приложение»
5. Запомните **ID приложения** (цифры в адресной строке или в настройках)

### Шаг 2. Получить Access Token

1. Откройте в браузере такую ссылку, подставив свой APP_ID:

```
https://oauth.vk.com/authorize?client_id=ВАШ_APP_ID&display=page&scope=wall,groups,offline&response_type=token&redirect_uri=https://oauth.vk.com/blank.html
```

Замените `ВАШ_APP_ID` на цифры из шага 1.

2. Разрешите приложению доступ к вашему аккаунту
3. После этого адресная строка будет выглядеть так:

```
https://oauth.vk.com/blank.html#access_token=ТУТ_ДЛИННАЯ_СТРОКА&expires_in=0&user_id=123456
```

4. Скопируйте всё, что идёт после `access_token=` и до следующего `&` — это ваш **токен**

### Шаг 3. Узнать ID группы

1. Откройте вашу группу ВКонтакте
2. Посмотрите на URL:
   - Если адрес `https://vk.com/club123456789` — ID группы: **123456789**
   - Если адрес `https://vk.com/mygroup` — зайдите в **Управление → Настройки**, там будет строка «ID группы: 123456789»

### Шаг 4. Вставить данные в app.py

Откройте файл `app.py` и найдите строки:

```python
VK_ACCESS_TOKEN = ""   # <- вставьте токен
VK_GROUP_ID = ""       # <- вставьте ID группы
```

Пример:
```python
VK_ACCESS_TOKEN = "vk1.a.AbCdEfGhIjKlMnOpQrStUvWxYz1234567890"
VK_GROUP_ID = "123456789"
```

Перезапустите сервер — кнопки «Опубликовать» и «Отложить» станут активны.

### Что может пойти не так

| Ошибка | Причина | Решение |
|---|---|---|
| Invalid access token | Токен неверный или истёк | Получите новый по ссылке из шага 2 |
| Group access token required | У токена нет прав на группы | Убедитесь, что в scope есть `groups` |
| Access denied | Вы не админ группы | Добавьте себя в администраторы группы |

### Какие права нужны

В ссылке из шага 2 параметр `scope=wall,groups,offline` означает:
- `wall` — публикация на стене
- `groups` — управление группами
- `offline` — токен не истекает (бессрочный)
